POR Hunting Auto Fast v16 설치 방법

1. Python 3.11 설치
   - 설치할 때 Add Python to PATH 체크

2. 압축 해제

3. 설치.bat 더블클릭

4. 실행.bat 더블클릭

5. 처음 실행 시 왼쪽 OpenDART API Key 입력
   - 한 번 입력하면 data/dart_api_key.txt에 자동 저장됩니다.

주의
- 인터넷 연결이 필요합니다.
- 다른 컴퓨터에서는 API Key를 한 번 다시 입력해야 합니다.
